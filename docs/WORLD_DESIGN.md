> **작성일**: 2026-02-25
> **상태**: 설계 확정 대기 (코드 작업 전 기획 문서)
> **목적**: 5탭 전체에 동물의숲+주토피아 세계관을 통합 적용하기 위한 마스터 설계서

## 1. 세계관 개요

### 1-1. 스토리

> **발른 마을**은 전설의 투자 거장들이 동물로 환생해 사는 작은 마을입니다.
> 유저는 이 마을에 새로 이사 온 **주민**이 됩니다.
> 매일 마을 광장에서 소식을 읽고, 병원에서 건강검진을 받고,
> 카페에서 이웃과 수다를 떨며, 시청에서 마을 일을 처리합니다.
> 거장 동물들에게 배우며, 자신만의 투자 기준을 세우고,
> 마을을 함께 번영시키는 것이 목표입니다.

### 1-2. 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **교육을 감싸기** | 투자 교육이지만, 유저는 "마을 생활"을 하는 느낌 |
| **안심을 팔기** | 시장 급락 = 마을에 비가 오지만, 구루들이 우산을 씌워줌 |
| **귀여움이 무기** | 캐릭터에 감정 이입 → 매일 들어오는 이유가 됨 |
| **일관된 톤** | 5탭 어디를 가도 "같은 마을에 있다"는 느낌 |

### 1-3. 마을 지도 (탭 = 장소)

```

│              발른 마을 전경               │
│                                         │

```

## 3. 탭별 세계관 적용 상세

### 3-1. 오늘 탭 → "마을 광장" 🏤

**장소 설정**: 마을 중심에 있는 광장. 게시판, 우체통, 벤치가 있음.

**UI 전환 맵핑:**

| 현재 요소 | 세계관 적용 | 구현 방식 |
|----------|-----------|---------|
| Pulse 요약 | 마을 날씨 | 건강 A=☀️맑음, B=🌤️, C=⛅, D=🌧️, F=⛈️ |
| 맥락 카드 | 마을 신문 | 구루가 배달. "🦉 버핏이 오늘 소식을 가져왔습니다" |
| 예측 투표 | 마을 투표소 | 투표함 UI + "주민 여러분의 의견을 모읍니다" |
| 복기 | 어제 결과 발표 | "어제 투표 결과: 🦊 캐시우드가 맞혔네요!" |
| 또래 비교 | 이웃 주민 동향 | "같은 동네 주민들은 평균 이만큼 갖고 있어요" |
| 출석 보상 | 광장 출근 도장 | "오늘도 마을에 오셨네요! +2 도토리" |

**톤 예시:**
- 기존: "총 자산 1.2억 | 건강 A | +0.3%"
- 변경: "☀️ 마을 날씨: 맑음 | 자산 1.2억 | 어제보다 +0.3% 성장"

**구루 배달 시스템:**
- 매일 아침 랜덤 구루 1명이 맥락 카드 배달 역할
- 시장 상황에 맞는 구루 선정 (급락 시 🐢 막스, 기술주 급등 시 🦊 캐시우드)
- 배달 구루의 한마디가 카드 상단에 표시

### 3-2. 분석 탭 → "마을 병원" 🏥

**장소 설정**: 마을 외곽에 있는 작은 병원. 🐢 거북이 막스가 주치의.

**왜 막스인가?**:
- 하워드 막스 = "2차 사고" = 진단에 가장 적합한 캐릭터
- 거북이 = 느리지만 정확 = 건강검진 이미지
- "급하게 행동하지 마세요, 먼저 검진받으세요"

**UI 전환 맵핑:**

| 현재 요소 | 세계관 적용 | 톤 |
|----------|-----------|-----|
| 건강 점수 A~F | 진단 결과 | "오늘 주민님의 건강 등급은 B입니다" |
| AI 진단 | 🐢 막스의 소견서 | "포트폴리오를 살펴봤는데요..." |
| 처방전 | 막스 선생님의 처방전 | "이렇게 조정하면 더 건강해질 거예요" |
| 리밸런싱 제안 | 약 처방 | "이 약(=리밸런싱)을 드시면~" |
| 코스톨라니 달걀 | 마을 기상대 | "지금 경제 계절은 '가을'이에요" |
| 위험 요소 | 주의 소견 | "이 부분이 좀 걱정됩니다..." |

**캐릭터 표정 연동:**
- 건강 A: 🐢 밝은 표정 "아주 건강하시네요!"
- 건강 C: 🐢 걱정 표정 "좀 신경 써야겠어요..."
- 건강 F: 🐢 심각 표정 "긴급 처방이 필요합니다"

**진료 흐름 UX:**
```
[병원 입구 일러스트]
     ↓
🐢 "안녕하세요, 정기검진 시간이에요"
     ↓
[검진 결과 카드] (건강 점수 + 6팩터)
     ↓
🐢 "소견서를 작성했어요" (AI 진단)
     ↓
[처방전] (리밸런싱 제안)
     ↓
🐢 "다음 검진은 내일 이 시간에!"
```

### 3-4. 라운지 탭 → "구루 카페" ☕

**장소 설정**: 마을 중심가 한켠에 있는 아늑한 카페. 모든 주민이 모이는 소셜 공간.

#### 핵심 기능

**A. 구루 방문 이벤트**
- 매일 랜덤 1~2명의 구루가 카페에 "방문"
- 방문 중인 구루에게 질문하면 AI가 그 톤으로 답변
- 방문 시간: 2시간 한정 → 희소성 → "지금 들어가야 해!"
- 알림: "🦉 버핏이 카페에 도착했습니다" (푸시)

**B. 구루 태그 (@멘션)**
- 글 작성 시 @버핏, @달리오 등 태그 가능
- AI가 해당 구루 톤으로 코멘트 자동 생성
- 무료: 1회/일, Premium: 3회/일
- 예: "@버핏 삼성전자 지금 사도 될까요?"
  → 🦉 "자네, 좋은 기업을 좋은 가격에 사는 게 핵심이지. 지금이 좋은 가격인지는 자네가 판단해야 해."

**C. 테이블 시스템 (주제별 분류)**
- 기존 게시판 카테고리를 "카페 테이블"로 리브랜딩
- 각 테이블에 최대 인원 표시 (활성 유저 수)

| 테이블 | 아이콘 | 설명 |
|--------|--------|------|
| 오늘 시장 | 📈 | 당일 시장 이야기 |
| 초보 질문 | 🌱 | 투자 입문자 전용 |
| 종목 토론 | 🔍 | 개별 종목 분석 |
| 자유 잡담 | 💬 | 투자 외 일상 |
| VIP 라운지 | 👑 | Premium 전용 |

**D. 주민 아바타 시스템**
- 가입 시 8종 동물 중 선택 (구루 10종과 겹치지 않는 동물)
- 활동에 따라 악세서리 해금 (모자, 안경, 넥타이 등)

| 선택 가능 동물 | 이모지 | 성격 이미지 |
|---------------|--------|-----------|
| 고양이 | 🐱 | 독립적, 관망형 |
| 강아지 | 🐶 | 활발, 소셜형 |
| 토끼 | 🐰 | 빠른 판단, 트레이더형 |
| 다람쥐 | 🐿️ | 저축형, 안정 추구 |
| 펭귄 | 🐧 | 차분, 분석형 |
| 판다 | 🐼 | 여유, 장기투자형 |

| 너구리 | 🦝 | 호기심, 탐험형 |

**E. 리액션 시스템**
- 기존 좋아요 → 동물 표정 리액션으로 변경

| 리액션 | 의미 | 아이콘 |
|--------|------|--------|
| 동의해요 | 좋아요 | 🐾👍 |

| 계정 삭제 | 전출 신고 | ⚠️ |

## 4. 마을 번영도 시스템

### 4-1. 개요

유저의 활동이 마을의 시각적 상태에 반영됩니다.
**모든 탭의 활동이 하나의 번영도로 수렴** → 탭 간 연결고리.

### 4-2. 번영도 레벨

| 레벨 | 이름 | 번영도 | 마을 상태 |
|------|------|--------|---------|
| 1 | 황무지 | 0~99 | 빈 땅, 잡초, 구루들 표정 어두움 |
| 2 | 작은 마을 | 100~299 | 기본 건물, 길 생김, 가로등 몇 개 |
| 3 | 성장하는 마을 | 300~599 | 꽃밭, 분수대, 카페 오픈 |
| 4 | 번영한 마을 | 600~999 | 큰 건물들, 축제 장식, 구루들 활발 |
| 5 | 전설의 마을 | 1000+ | 황금빛, 특별 이펙트, 전설 건물 |

### 9-2. 의존성

```
번영도 시스템
  ├── 오늘 탭 활동 → 번영도 변경
  ├── 분석 탭 활동 → 번영도 변경
  ├── 마을 탭 활동 → 번영도 변경 (이미 있음)
  ├── 라운지 탭 활동 → 번영도 변경
  └── 마을 탭 시각 반영 ← 번영도 읽기

구루 캐릭터 (이미 구현)
  ├── 오늘 탭 ← 캐릭터 아바타 import
  ├── 분석 탭 ← 막스 캐릭터 import
  ├── 마을 탭 ← 전체 캐릭터 (이미 있음)
  └── 라운지 탭 ← 캐릭터 코멘트 + 아바타
```

### 9-3. 작업 순서 제안

```
Phase 1: 연결 (1주)
  ├── 오늘 탭에 구루 캐릭터 한 줄 추가
  ├── 분석 탭에 🐢 막스 아바타 추가
  ├── 전체 탭 섹션명 리네이밍
  └── locales 세계관 텍스트 추가

Phase 2: 라운지 고도화 (1주)
  ├── 카페 배경 + 테이블 시스템
  ├── 구루 태그 (@멘션) AI 응답
  ├── 구루 방문 이벤트
  └── 리액션 시스템

Phase 3: 번영도 (1주)
  ├── useVillageProsperity 훅
  ├── 전 탭 활동 → 번영도 연동
  ├── 마을 시각 반영
  └── 부재 패널티 + 복귀 이벤트

Phase 4: 유저 아바타 (1주)
  ├── 아바타 선택 UI
  ├── 온보딩 통합
  ├── 라운지 프로필 연동
  └── 악세서리 해금 시스템

Phase 5: 이벤트 (이후)
  ├── 시장 연동 날씨
  ├── 계절 이벤트
  └── 한정 아이템
```

## 15. 구루별 캐릭터 디테일 시트

> 주토피아 수준의 디테일: 동물 특성이 투자 스타일과 연결

### 🦉 워렌 버핏 (올빼미)

**왜 올빼미?**: 밤에도 보는 넓은 시야 = 장기 투자 안목. 고개를 270도 돌림 = 다양한 관점.

| 항목 | 디테일 |
|------|--------|
| **카페 주문** | 체리코크 (실제 버핏이 좋아하는 음료) |
| **집 인테리어** | 오래된 서재, 신문 더미, 코카콜라 캔 |
| **산책 스타일** | 느긋하게 걷다가 갑자기 멈춰서 뭔가 관찰 |
| **잠버릇** | 일찍 자고 일찍 일어남 (5AM 기상) |
| **좋아하는 날씨** | 폭풍 (☀️ 보다 ⛈️ 때 더 활발) — "공포에 사라" |
| **입버릇** | "자네~", "장기적으로 보면~", "모트(Moat)가 있나?" |
| **숨겨진 약점** | 기술주 이야기하면 약간 불편해함 |
| **린치와의 케미** | 린치가 찾아오면 의자 하나 내어줌 |
| **캐시우드와의 케미** | 만나면 눈을 가늘게 뜸 (존중하지만 동의 안 함) |

### 🦌 레이 달리오 (사슴)

**왜 사슴?**: 경계심 높은 초식동물 = 리스크 감지. 무리 지어 이동 = 분산 투자.

| 항목 | 디테일 |
|------|--------|
| **카페 주문** | 명상차 (허브티) |
| **집 인테리어** | 깔끔한 사무실, 화이트보드에 원칙 목록 |
| **산책 스타일** | 정해진 루트를 매일 같은 시간에 걸음 (원칙적) |
| **잠버릇** | 명상 후 취침 (정확히 10PM) |
| **좋아하는 날씨** | 맑은 날 (데이터가 깨끗할 때) |
| **입버릇** | "원칙적으로~", "경제 기계는~", "사이클을 보면~" |
| **숨겨진 약점** | 가끔 너무 원칙적이라 유연하지 못함 |
| **로저스와의 케미** | 글로벌 데이터 교환하며 산책 |

### 🦊 캐시 우드 (여우)

**왜 여우?**: 영리하고 빠른 판단 = 혁신 투자. 호기심 많음 = 새로운 기술 탐구.

| 항목 | 디테일 |
|------|--------|
| **카페 주문** | 더블 에스프레소 (에너지 넘침) |
| **집 인테리어** | 미래지향 — 홀로그램 스크린, 테슬라 미니어처 |
| **산책 스타일** | 빠르게 걸으며 모두에게 말 걸기 |
| **잠버릇** | 늦게 자고 늦게 일어남 (미국 시장 시간) |
| **좋아하는 날씨** | 무지개 (변화의 상징) |
| **입버릇** | "5년 뒤를 봐요!", "파괴적 혁신!", "AI가!" |
| **숨겨진 약점** | 단기 실적이 안 좋으면 살짝 풀 죽음 |
| **머스크와의 케미** | 같이 미래 기술 이야기하면 시간 가는 줄 모름 |
| **버핏과의 케미** | "선배님, 시대가 바뀌었어요!" 하면 버핏이 한숨 |

### 🐆 드러킨밀러 (치타)

**왜 치타?**: 가장 빠른 동물 = 추세 전환 포착. 폭발적 스프린트 + 기다림.

| 항목 | 디테일 |
|------|--------|
| **카페 주문** | 블랙 커피 (군더더기 없음) |
| **집 인테리어** | 다중 모니터, 차트 빼곡, 미니멀 |
| **산책 스타일** | 빠르게 걷다가 갑자기 멈춤 (추세 전환점처럼) |
| **좋아하는 날씨** | 바람 부는 날 (변화의 조짐) |
| **입버릇** | "추세가 바뀌는 순간!", "집중하자", "지금이야" |
| **숨겨진 약점** | 머스크의 장난에 쉽게 짜증냄 |

### 🐺 마이클 세일러 (늑대)

**왜 늑대?**: 무리의 리더 = 비트코인 맥시멀리스트. 울부짖음 = 강렬한 확신.

| 항목 | 디테일 |
|------|--------|
| **카페 주문** | 안 마심 (비트코인 이야기만 하러 옴) |
| **집 인테리어** | 비트코인 포스터, 오렌지색 조명, 레이저 눈 거울 |
| **산책 스타일** | 비트코인 가격 오르면 돌아다니고, 내리면 집에 틀어박힘 |
| **좋아하는 날씨** | 폭풍 후 무지개 (급락 후 반등) |
| **입버릇** | "비트코인 외에 뭐가 있어?", "44조 시간!", "HODL" |
| **다이먼과의 케미** | 만나면 30초 안에 논쟁 시작 |

### 🦁 제이미 다이먼 (사자)

**왜 사자?**: 금융계의 왕 = 전통 금융 수장. 위엄 있지만 가끔 으르렁.

| 항목 | 디테일 |
|------|--------|
| **카페 주문** | 클래식 아메리카노 (전통적) |
| **집 인테리어** | 고급 사무실, 갈색 가죽 소파, 금고 |
| **산책 스타일** | 당당하게 걸음 (다른 구루들이 길을 비켜줌) |
| **좋아하는 날씨** | 맑은 날 (안정적인 시장) |
| **입버릇** | "기본에 충실", "리스크 관리가 먼저", "규제가 필요해" |
| **세일러와의 케미** | "그 비트코인이란 게..." 하면 세일러가 달려옴 |

### 🦎 일론 머스크 (카멜레온)

**왜 카멜레온?**: 색이 바뀜 = 예측 불가능. 적응력 = 우주/자동차/AI 다방면.

| 항목 | 디테일 |
|------|--------|
| **카페 주문** | 매번 다름 (한 번은 커피, 한 번은 주스, 한 번은 물) |
| **집 인테리어** | 로켓 모형, 테슬라 미니, 밈 포스터 |
| **산책 스타일** | 불규칙 — 갑자기 나타났다 사라짐 |
| **좋아하는 날씨** | 모든 날씨 (상관 안 함) |
| **입버릇** | "ㅋㅋ", "제가 그냥 하면 되는데?", "화성 갈 사람?" |
| **드러킨밀러와의 케미** | 일부러 놀리러 감 → 드러킨밀러 짜증 |

### 🐻 피터 린치 (곰)

**왜 곰?**: 숲에서 열매를 찾음 = 일상에서 종목 발굴. 친근하고 덩치 큼.

| 항목 | 디테일 |
|------|--------|
| **카페 주문** | 꿀차 (곰이니까) |
| **집 인테리어** | 마트 전단지 컬렉션, 소비재 제품 진열 |
| **산책 스타일** | 느긋하게 걸으며 가게 하나하나 관찰 |
| **좋아하는 날씨** | 봄 (새 제품이 나오는 계절) |
| **입버릇** | "마트에서 봤는데~", "PEG 비율이~", "생활 속에 답이 있어" |
| **버핏과의 케미** | 존경해서 버핏이 뭐 하면 따라가서 엿봄 |

### 🐢 하워드 막스 (거북이)

**왜 거북이?**: 느리지만 정확 = 신중한 진단. 등껍질 = 방어적 투자.

| 항목 | 디테일 |
|------|--------|
| **카페 주문** | 따뜻한 차 (천천히 마심) |
| **집 인테리어** | 병원 겸 서재, 사이클 차트, 메모 가득 |
| **산책 스타일** | 매우 천천히, 주변을 세밀히 관찰 |
| **좋아하는 날씨** | 안개 (불확실성 속에서 진실을 찾는 것) |
| **입버릇** | "2차 사고를 해봐", "사이클이 있어", "모두가 낙관할 때..." |

## 23. "매일 다른 마을" — 반복감 제거 시스템

> **핵심 원리**: 유저가 1주일 플레이해도 "에이 똑같네" 느끼지 않으려면,
> 매일 보는 것의 **조합**이 달라야 한다. 하나하나는 단순해도 조합하면 수천 가지.

### 23-1. 변동 레이어 (6겹)

매일 마을 화면을 구성하는 6개 변수가 독립적으로 바뀜:

```
오늘의 마을 = 시간 × 날씨 × 시장 × 요일이벤트 × 구루무드 × 유저상태
             24종   5종    5종     7종          10명×8무드    6종

이론적 조합: 24 × 5 × 5 × 7 × 80 × 6 = 2,016,000 가지
→ 매일 같은 화면을 볼 확률 ≈ 0%
```

| 레이어 | 변동 요소 | 소스 |
|--------|---------|------|
| **1. 시간** | 시간대별 하늘/조명/BGM (이미 구현) | 기기 시계 |
| **2. 날씨** | 실제 날씨 + 시장 날씨 혼합 | OpenWeather API + 시장 데이터 |
| **3. 시장 무드** | 지수 등락에 따른 마을 분위기 | 코스피/S&P/BTC |
| **4. 요일 이벤트** | 요일별 구루 스케줄 (섹션 11) | 고정 스케줄 |
| **5. 구루 개별 상태** | 10명 각각의 기분/활동/옷 | 시장+날씨+랜덤 |
| **6. 유저 상태** | 스트릭/우정/번영도에 따른 반응 | 유저 데이터 |

### 23-2. 일주일 체험 시뮬레이션

**유저 "김투자"의 1주일:**

#### 월요일 (맑음, 코스피 +0.5%)
```

🐻 린치가 마을 상점 앞에서 뭔가 관찰 중
🦎 머스크는 아직 안 보임 (늦잠)
맥락 카드: 🦌 달리오가 배달 "이번 주 거시경제 전망을 정리했습니다"
```

#### 화요일 (비, 코스피 -1.2%)
```
🌧️ 비 내리는 오후. 마을에 빗소리.
구루 전원 우산 쓰고 있음 (날씨 연동)
🐢 막스가 병원 앞에서 "걱정 마세요" 팻말 들고 서 있음
🐺 세일러가 비를 맞으며 혼자 걸어다님 (비트코인도 하락)
🦊 캐시우드 집에 불이 꺼져 있음 (기술주 하락 → 풀이 죽음)
🐻 린치가 카페에서 "마트에서 찾은 종목" 이야기 중 (화요일 이벤트)
맥락 카드: 🐢 막스가 배달 "하락의 맥락을 설명드립니다"
```

#### 수요일 (흐림, 코스피 +0.1%)
```
⛅ 흐린 오전. 평온한 분위기.
🦊 캐시우드가 광장에서 "혁신 수요일" 특별 강연 (수요일 이벤트)
🦉 버핏이 캐시우드 강연을 멀리서 팔짱 끼고 지켜보는 중
🦎 머스크가 캐시우드 옆에서 "맞아맞아!" 하고 있음
🐆 드러킨밀러가 차트 보면서 혼자 산책 중
카페에서 구루 2명이 커피 마시며 대화 중 (랜덤 조합)
```

#### 목요일 (눈, 코스피 -2.8% 급락)
```
⛈️ 폭풍! 마을이 어두움. 긴장감.
구루 전원 긴급 회의 중 (광장에 모여 있음)
🦉 버핏만 웃고 있음 ("공포에 사라")
🐢 막스가 병원에서 "리스크 관리 교실" 긴급 개최 (목요일 이벤트 + 급락 트리거)
🐺 세일러가 집에 틀어박힘 (비트코인 동반 하락)
🦁 다이먼이 마을 중앙에서 "침착하세요" 연설 중
맥락 카드: 위기 특별판 — 구루 3명의 관점 동시 제공
편지 도착: 🦌 달리오 "1987년에도 이런 날이 있었습니다"
```

#### 금요일 (맑아짐, 코스피 +1.5% 반등)
```
🌈 무지개! 마을에 안도 분위기.
🦉 버핏이 카페에서 "주간 복기의 밤" 준비 중 (금요일 이벤트)
🦊 캐시우드가 다시 밝은 표정으로 돌아다님
🐺 세일러가 집에서 나와 깃발 흔들기 시작
🐯 로저스가 "이머징 마켓에도 좋은 신호!" 하며 돌아다님
카페 저녁: 구루 5명+ 모여서 이번 주 복기 토론
```

#### 토요일 (더움 30°C, 시장 휴장)
```
☀️ 뜨거운 토요일. 구루들 여름옷 착용.
🐯 로저스가 "글로벌 탐험" 특별 이벤트 (토요일 이벤트)
🦎 머스크가 선글라스 쓰고 아이스크림 먹으며 돌아다님
🦉 버핏이 나무 그늘에서 낮잠 (시장 휴장 → 여유)
🐻 린치가 해변(?)에서 서핑 시도 중 (토요일 특별 활동)
마을 분위기: 평화로움, 주민들 산책 많음
```

#### 일요일 (선선함 18°C)
```
🌤️ 선선한 일요일. 가을 느낌 재킷 착용.
🦎 머스크의 "자유의 날" — 깜짝 이벤트 (일요일 이벤트)
   → 이번 주: 마을에 불꽃놀이 설치 (다음 주: 뭐가 될지 모름)
구루들이 삼삼오오 산책 중 (릴랙스)
🦌 달리오가 명상하는 모습 (일요일 루틴)
🐢 막스가 서재에서 책 읽는 중
카페에서 주민 모임 (라운지 활발)
내일 예고: "월요일 달리오 주간 브리핑 준비 중..."
```

**→ 같은 주는 없다. 시장이 바뀌고, 날씨가 바뀌고, 구루 조합이 바뀌니까.**

## 24. 구루 활동 시스템 (Activity System)

> "걸어다니기만 하면 NPC가 아니라 배경이다."
> 구루가 다양한 활동을 해야 "살아있다"고 느낀다.

### 24-1. 활동 종류 (20가지+)

#### 일상 활동

| 활동 | 애니메이션 | 트리거 |
|------|----------|--------|
| 🚶 산책 | 좌우 이동 | 기본 (현재 구현) |
| 💤 낮잠 | 눈 감고 "zzz" 파티클 | 시장 휴장 OR 오후 2~3시 |
| 📖 독서 | 책 들고 앉아있음 | 랜덤 (버핏/막스/달리오 주로) |
| ☕ 커피 | 컵 들고 마시는 모션 | 카페 근처 + 아침 시간 |
| 💬 수다 | 두 구루가 마주보고 말풍선 | 같은 파벌 OR 라이벌 근처일 때 |
| 🎵 콧노래 | 음표 파티클 | 시장 상승 + 좋은 날씨 |
| 📱 뭔가 보는 중 | 아래 쳐다보며 스크롤 | 머스크 전용 (트위터) |
| 🧘 명상 | 가부좌 + 눈 감음 | 달리오 전용 (아침/일요일) |

#### 날씨/기온 활동

| 활동 | 애니메이션 | 트리거 |
|------|----------|--------|
| ☂️ 우산 쓰기 | 우산 들고 이동 | 비 올 때 |
| 🏄 서핑 | 파도 위 보드 | 30°C+ 주말 (린치/로저스) |
| 🍦 아이스크림 | 아이스크림 핥기 | 28°C+ |
| ❄️ 벌벌 떨기 | 몸 떨림 + 입김 | 0°C 이하 |
| 🧣 목도리 | 목도리 착용 이동 | 5°C 이하 |
| ☀️ 선글라스 | 선글라스 착용 | 맑음 + 25°C+ |
| 🌸 꽃구경 | 꽃 앞에서 멈춤 | 봄 (3~4월) |
| 🍂 낙엽 밟기 | 낙엽 파티클 | 가을 (10~11월) |

#### 감정 활동

| 활동 | 애니메이션 | 트리거 |
|------|----------|--------|
| 💃 춤추기 | 좌우 흔들 + 점프 | 담당 분야 급등 (+3%+) |
| 🎉 축하 | 박수 + 꽃가루 파티클 | 유저 예측 적중/뱃지 해금 |
| 😢 풀 죽음 | 고개 숙이고 느린 이동 | 담당 분야 급락 (-3%+) |
| 😤 논쟁 | 팔 흔들며 마주보기 | 라이벌 만남 + 관련 이슈 |
| 🤗 포옹 | 두 구루 가까이 + 하트 | 동맹 + 시장 급등 |
| 🏃 달리기 | 빠른 이동 | 드러킨밀러 전용 (추세 전환 감지) |
| 😴 졸기 | 고개 끄덕끄덕 + zzz | 밤 시간 (22시+) |
| 📣 연설 | 연단 위에서 | 요일 이벤트 담당 구루 |

### 24-2. 활동 결정 알고리즘

```
매 30초마다 각 구루에 대해:

1. 시간 체크 → 잠자는 시간이면 → 💤 수면
2. 요일 이벤트 체크 → 해당 요일 담당이면 → 📣 이벤트 활동
3. 날씨/기온 체크 → 조건 맞는 활동 후보 추출
4. 시장 무드 체크 → 감정 활동 후보 추출
5. 주변 구루 체크 → 라이벌/동맹 근처면 → 💬 수다 OR 😤 논쟁
6. 위 후보 중 가중 랜덤 선택 (같은 활동 연속 방지)
7. 20% 확률로 "순수 랜덤" 활동 (예측 불가능성 추가)
```

### 24-3. 구루별 활동 확률 가중치

| 구루 | 자주 하는 활동 | 거의 안 하는 활동 |
|------|-------------|---------------|
| 🦉 버핏 | 독서(30%), 낮잠(20%), 커피(15%) | 달리기, 서핑 |
| 🦌 달리오 | 명상(25%), 산책(20%), 독서(15%) | 춤, 서핑 |
| 🦊 캐시우드 | 수다(25%), 춤(15%), 달리기(10%) | 낮잠, 명상 |
| 🐆 드러킨밀러 | 달리기(20%), 독서(15%), 혼자산책(20%) | 수다, 춤 |
| 🐺 세일러 | 연설(20%), 논쟁(15%), 깃발흔들기(15%) | 명상, 독서 |
| 🦁 다이먼 | 산책(25%), 커피(20%), 독서(15%) | 서핑, 춤 |
| 🦎 머스크 | 스마트폰(25%), 아이스크림(15%), 깜짝행동(20%) | 명상, 독서 |
| 🐻 린치 | 관찰(30%), 서핑(10%), 커피(15%) | 명상 |
| 🐢 막스 | 독서(30%), 느린산책(25%), 차마시기(15%) | 달리기, 서핑 |
| 🐯 로저스 | 빠른산책(25%), 서핑(10%), 수다(15%) | 낮잠 |

## 25. 실시간 날씨 연동 시스템

> "앱 열었는데 밖에 비 오고 있고, 마을에도 비가 오고 있으면 — 소름."

### 25-1. 날씨 데이터 소스

```
실제 날씨 (OpenWeather API, 유저 위치 기반)
    ↓
마을 날씨 = 실제 날씨 × 시장 무드 블렌딩
    ↓
구루 옷차림 + 활동 + 배경 변경
```

**블렌딩 규칙:**
- 기본: 실제 날씨 70% + 시장 무드 30%
- 시장 급변 시 (-3%+ or +3%+): 시장 무드 70% + 실제 날씨 30%
- 예: 밖에 맑은데 시장 급락 → 마을은 흐리고 바람 분다

### 25-2. 날씨 → 마을 비주얼

| 실제 날씨 | 마을 비주얼 | 파티클 효과 |
|----------|-----------|-----------|
| ☀️ 맑음 | 밝은 파란 하늘, 구름 없음 | 빛 반짝임 파티클 |
| ⛅ 구름 | 구름 2~3개 떠다님 | 구름 그림자 이동 |
| 🌧️ 비 | 하늘 회색, 물웅덩이 | 빗방울 파티클 + 물결 |
| ❄️ 눈 | 하얀 지면, 나뭇가지 눈 | 눈송이 파티클 (천천히 내림) |
| 🌫️ 안개 | 반투명 안개 레이어 | 안개 드리프트 효과 |
| ⛈️ 폭풍 | 어두운 하늘, 번개 | 번개 플래시 + 빗소리 강화 |
| 🌈 무지개 | 비 그친 후 아치형 무지개 | 색상 파티클 |

### 25-3. 기온 → 구루 옷차림

| 기온 범위 | 옷차림 레벨 | 설명 |
|----------|-----------|------|
| -10~0°C | 패딩 + 귀마개 + 장갑 | 풀 겨울 장비 |
| 0~10°C | 코트 + 목도리 | 겨울 |
| 10~18°C | 재킷 + 긴팔 | 봄/가을 |
| 18~25°C | 기본 의상 | 쾌적 (SVG 기본) |
| 25~30°C | 반팔 + 선글라스 | 여름 |
| 30°C+ | 민소매 + 모자 + 부채 | 폭염 |

**구루별 옷차림 디테일:**

| 구루 | 겨울 특징 | 여름 특징 |
|------|---------|---------|
| 🦉 버핏 | 갈색 카디건 + 머플러 | 체크 셔츠 반팔 |
| 🦌 달리오 | 네이비 더플코트 | 린넨 셔츠 |
| 🦊 캐시우드 | 빨간 코트 | 화이트 블라우스 |
| 🐆 드러킨밀러 | 블랙 패딩 | 블랙 티셔츠 |
| 🐺 세일러 | 비트코인 오렌지 패딩 | 비트코인 오렌지 티셔츠 (세일러는 항상 오렌지) |
| 🦁 다이먼 | 정장 코트 | 정장 (여름에도 정장 — 은행장이니까) |
| 🦎 머스크 | 우주복 파카 | 하와이안 셔츠 (매번 다른 패턴) |
| 🐻 린치 | 플리스 조끼 | 마트 앞치마 (여름에도 종목 찾으러) |
| 🐢 막스 | 울 스웨터 | 흰 가운 (병원이니까 — 여름에도) |
| 🐯 로저스 | 모험가 재킷 + 모자 | 탐험가 반팔 + 반바지 |

### 25-4. 계절 마을 변화

| 계절 | 기간 | 마을 변화 |
|------|------|---------|
| **봄** | 3~5월 | 벚꽃 나무, 새 둥지, 꽃밭 만개, 풀 연두색 |
| **여름** | 6~8월 | 진한 초록, 매미 소리, 분수대 활발, 해변 오픈 |
| **가을** | 9~11월 | 단풍, 낙엽 파티클, 호박 장식, 갈색 톤 |
| **겨울** | 12~2월 | 눈 쌓인 지붕, 벽난로 연기, 크리스마스 조명 |

**전환 효과:**
- 계절이 바뀌는 첫 날: "마을에 봄이 왔습니다!" 배너
- 1주에 걸쳐 점진적 변화 (한 번에 바뀌지 않음)
- 나무에 잎이 하루에 몇 장씩 떨어지다가 다 떨어지는 식

## 26. 구루 감정 엔진 (Mood Engine)

> 단순히 4표정(bullish/bearish/cautious/neutral)에서 → 8감정 + 블렌딩

### 26-1. 확장된 감정 (8가지)

| 감정 | 표정 | 트리거 |
|------|------|--------|
| 😊 기쁨 (JOY) | 눈 반달, 입꼬리 올라감 | 담당 분야 상승 |
| 😤 흥분 (EXCITED) | 눈 빛남, 입 크게 벌림 | 담당 분야 급등 (+3%+) |
| 😌 평온 (CALM) | 편안한 눈, 미소 | 시장 보합 + 좋은 날씨 |
| 🤔 고민 (THINKING) | 눈 약간 찡그림, 입 삐죽 | 시장 불확실 |
| 😟 걱정 (WORRIED) | 눈썹 내려감, 입 굳음 | 담당 분야 하락 |
| 😢 슬픔 (SAD) | 눈 처짐, 땀방울 | 담당 분야 급락 (-3%+) |
| 😠 분노 (ANGRY) | 눈 날카로움, 이빨 | 라이벌과 논쟁 중 |
| 😴 졸림 (SLEEPY) | 눈 반쯤 감김, 하품 | 밤 시간 OR 시장 휴장 |

### 26-2. 감정 결정 공식

```
구루_감정 = f(시장_섹터, 전체_시장, 날씨, 시간, 요일, 주변_구루, 유저_관계)

예시: 🦊 캐시우드 오늘의 감정 계산
- 나스닥 +2.1% → 기쁨 가중치 +40
- 테슬라 +5% → 흥분 가중치 +30
- 비 오는 날씨 → 기쁨 -10
- 오후 3시 (장 마감 직전) → 긴장 +10
- 옆에 버핏이 있음 → 약간 긴장 +5
- 결과: 흥분 60%, 기쁨 30%, 고민 10%
→ 표정: 😤 흥분 (지배적 감정)
→ 활동: 💃 춤추기 OR 📣 연설
```

### 26-3. 구루별 감정 민감도

| 구루 | 시장에 민감 | 날씨에 민감 | 다른 구루에 민감 |
|------|-----------|-----------|-------------|
| 🦉 버핏 | 낮음 (항상 침착) | 낮음 | 캐시우드에만 반응 |
| 🦌 달리오 | 중간 (원칙 기반) | 낮음 | 낮음 (독립적) |
| 🦊 캐시우드 | 높음 (감정적) | 중간 | 버핏/머스크에 반응 |
| 🐆 드러킨밀러 | 매우 높음 (트레이더) | 낮음 | 머스크에 짜증 |
| 🐺 세일러 | BTC에만 극도로 민감 | 없음 | 다이먼에 폭발 |
| 🦁 다이먼 | 중간 (은행장 체면) | 낮음 | 세일러에 한숨 |
| 🦎 머스크 | 불규칙 (예측 불가) | 높음 (날씨 좋으면 신남) | 모든 구루에 장난 |
| 🐻 린치 | 소비재에만 민감 | 높음 (봄이면 신남) | 버핏에 존경 표시 |
| 🐢 막스 | 중간 (사이클 관점) | 안개에만 반응 | 전체 관찰 |
| 🐯 로저스 | 원자재/이머징에 민감 | 높음 (더우면 활발) | 달리오와 친밀 |

### 26-4. 감정 전이 (구루 간 영향)

```
시나리오: 비트코인 +10%

🐺 세일러 → 😤 흥분 (즉시) → 마을 돌아다니며 깃발 흔듦
   ↓ 영향
🦎 머스크 → 😊 기쁨 (동맹이라 같이 흥분)
🦊 캐시우드 → 😊 기쁨 (혁신파 동맹)
   ↓ 대조
🦁 다이먼 → 😠 짜증 (라이벌이 기뻐하니까)
🦉 버핏 → 😌 평온 (비트코인에 관심 없음, 변화 없음)
🐢 막스 → 🤔 고민 ("이게 사이클의 어디쯤일까...")
```

## 27. 마을 미니 이벤트 (Daily Micro-Events)

> 큰 이벤트가 아니라 "앗 뭐야?" 하는 작은 발견들이 매일 달라야 한다.

### 27-1. 랜덤 미니 이벤트 풀 (30가지+)

매일 1~3개가 랜덤으로 발생:

**구루 간 상호작용:**
1. 버핏과 린치가 벤치에 나란히 앉아 신문 보기
2. 캐시우드가 머스크에게 뭔가 보여주며 흥분
3. 세일러가 다이먼에게 다가가자 다이먼이 도망
4. 달리오와 로저스가 지구본 보며 대화
5. 드러킨밀러가 막스에게 건강검진 받으러 옴
6. 머스크가 버핏에게 "테슬라 어때요?" → 버핏 한숨
7. 린치가 마을 상점 물건 하나하나 살펴보기
8. 막스와 버핏이 조용히 체스 두기
9. 로저스가 호랑이답게 나무에 스크래치 남기기
10. 전원이 광장에 모여 단체 사진 (매우 희귀, 1% 확률)

**날씨 미니 이벤트:**
11. 비 온 후 무지개 아래에서 구루들 셀카
12. 눈 오면 머스크가 눈사람 만들기
13. 폭염 시 린치가 수박 나눠주기
14. 안개 속에서 막스가 등불 들고 산책
15. 바람 불면 캐시우드 우산 날아감 → 드러킨밀러가 잡아줌

**유저 반응 이벤트:**
16. 유저가 100일 출석 → 구루 전원 줄서서 하이파이브
17. 예측 10연속 → 드러킨밀러 "와..." 감탄
18. 첫 Premium 가입 → 다이먼 "VIP 환영합니다" 인사
19. 카페 첫 글 → 린치 "새 이웃! 환영!" 말풍선
20. 집 레벨업 → 구루들이 집 앞에서 축하

**계절 미니 이벤트:**
21. 봄: 벚꽃 아래 구루 3명 피크닉
22. 여름: 해변에서 서핑 대회 (린치 vs 로저스)

25. 발렌타인: 라이벌 구루들이 서로 안 보는 척하며 카드 전달

**시장 미니 이벤트:**
26. 신고가 갱신: 마을에 축포 + 깃발
27. 금리 인상 발표: 달리오 긴급 강연
28. IPO 대형 상장: 캐시우드 흥분하며 돌아다님

30. 비트코인 반감기: 세일러 카운트다운 이벤트

## 30. 구루 뉴스 리액션 시스템 (Guru News Reactions)

> "같은 뉴스를 10명이 다르게 해석한다"
> — 이것이 투자의 핵심이고, 이 앱이 가르치는 것.
> 유저는 뉴스를 읽는 게 아니라 **구루들의 반응을 구경**한다.

### 30-1. 콘셉트

```
오늘의 뉴스 헤드라인
    ↓
Gemini가 10 구루 관점으로 각각 반응 생성
    ↓
뉴스 카드 + 구루 리액션 바 표시
    ↓
유저: "아 같은 뉴스인데 버핏이랑 캐시우드 반응이 이렇게 다르구나"
    ↓
투자 관점의 다양성을 자연스럽게 학습
```

### 30-2. 뉴스 카테고리 & 소스

| 카테고리 | 아이콘 | 소스 | 하루 표시 수 |
|----------|--------|------|------------|
| **주식** | 📈 | 코스피/코스닥 주요 종목 등락 | 1~2개 |
| **암호화폐** | ₿ | 비트코인/이더리움/알트코인 | 1개 |
| **경제** | 🏦 | 금리/CPI/고용/GDP/환율 | 1개 |
| **정치** | 🏛️ | 미국 정책/한국 정책/지정학 | 0~1개 |
| **글로벌** | 🌍 | 해외 시장/유럽/중국/일본 | 0~1개 |
| **합계** | | | **하루 3~5건** |

**뉴스 수집 방법:**
- `generate-context-card` Edge Function이 이미 시장 데이터 수집 중
- 여기에 **"오늘의 톱 뉴스 5건 요약"** 프롬프트 추가
- 3시간마다 갱신 (맥락 카드와 동일 주기)

|----------|---------|
| 금리 인상 | 🦌 달리오가 광장에서 긴급 브리핑 시작. 다른 구루들이 모여듦. |
| 비트코인 급등 | 🐺 세일러가 마을 돌아다니며 깃발 흔듦. 🦁 다이먼이 집으로 들어감. |
| 기술주 급락 | 🦊 캐시우드 집에 불 꺼짐. 🦉 버핏이 밖에서 미소. |
| 지정학 긴장 | 🐯 로저스가 지도 펼치고 설명 중. 🦁 다이먼 표정 굳음. |
| 실적 서프라이즈 | 🐻 린치가 해당 기업 로고 들고 "내가 마트에서 봤다고!" |

### 30-7. 무료 vs Premium

| 요소 | 무료 | Premium |
|------|------|---------|
| 뉴스 헤드라인 | 전체 (3~5건) | 전체 |
| 구루 반응 수 | 뉴스당 **2명** | 뉴스당 **전원 (10명)** |
| 반응 구루 선택 | 자동 (가장 관련 높은 2명) | 전원 |
| 내 포트폴리오 영향 | 요약 1줄 | 상세 분석 |
| 구루에게 질문 | ❌ | ✅ (내 스승 1명) |
| 감정 분포 차트 | ❌ | ✅ |

**Premium 전환 트리거:**
- 무료 유저가 2명 반응 보고 "나머지 8명은?" 궁금 → 🔒 잠금 아이콘
- "🐺 세일러와 🦁 다이먼의 반응이 궁금하다면..." → Premium 유도
- 특히 **라이벌 구루 반응**을 잠금 (버핏은 보이는데 캐시우드는 잠금 → 궁금)

### 31-7. 상점 소품 구매 시스템

**숨겨진 메커니즘:**
- 소품을 사면 → 해당 기업 뉴스를 우선 표시 (관심 종목 자동 등록)
- "나이키 운동화를 산 주민은 나이키 실적 발표일에 알림"
- 소품 = 관심 표현 = 개인화 데이터 = 더 나은 맥락 카드

### 31-8. 피터 린치 특별 연동 — "마트 투어"

> 린치의 철학: "일상에서 좋은 종목을 찾아라"
> 매주 린치가 상점 거리를 돌아다니며 "발견" 코멘트

```
매주 화요일 "린치의 마트 투어":

🐻 린치가 상점 거리를 걸으며:

발자국 운동화 앞: "이 운동화 요즘 줄 서서 사더라.
  매출이 오르겠는데? PEG 비율 확인해봐."

사과 상점 앞: "사과폰 신모델 나왔는데 다들 바꾸네.
  이런 게 생활 속 투자 신호야."

번개 마차 앞: "전기마차가 마을에 많아졌어.
  근데 경쟁자도 늘어서... 좀 더 봐야 해."
```

**교육 효과:**
- 유저가 자연스럽게 "일상 소비 → 투자 아이디어" 연결법을 배움
- 린치의 PEG 비율, 소비재 분석법을 실제 브랜드로 체험
- "아 나이키 운동화 줄 서는 거 보고 주식 사는 거구나" → 투자 감각
