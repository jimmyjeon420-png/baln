## 3. 스크린샷별 상세 가이드

### 스크린샷 1: 맥락 카드 (핵심 차별점)

**캡처 화면**: `app/(tabs)/index.tsx` (홈 탭) — 맥락 카드 섹션

**메시지 (상단 오버레이)**:
```
📈 오늘 시장이 왜 이렇게 움직였는지
4겹 맥락으로 이해하세요
```

**화면 구성**:
- 맥락 카드 1장 (레이어 1~4 표시)
- 예시 텍스트:
  - 레이어 1 (역사): "2008년 금융위기 당시 S&P500 -5% 하락 후 6개월 내 회복"
  - 레이어 2 (거시경제): "미국 CPI 3.1% 발표 → 금리 인상 우려 → 기술주 하락"
  - 레이어 3 (기관 행동): "외국인 3일 연속 순매도 ₩3조 (리밸런싱 시즌)"
  - 레이어 4 (내 포트폴리오): "당신의 포트폴리오는 -1.2% 영향, 건강 점수 B 유지"

**캡션 (App Store Description 하단)**:
```
📈 오늘 시장이 왜 이렇게 움직였는지
4겹 맥락으로 이해하세요
```

**촬영 팁**:
- ✅ 실제 데이터 사용 (가짜 데이터 X)
- ✅ 레이어 4개 모두 보이도록 스크롤 중간 위치
- ❌ 위기 알림 (-3% 급락) 사용 금지 (공포 마케팅)

### 스크린샷 2: 예측 게임 (습관 루프)

**캡처 화면**: `app/(tabs)/index.tsx` (홈 탭) — 예측 투표 섹션

**메시지 (상단 오버레이)**:
```
🎮 내일 시장 예측 → 복기 → 기준 형성
매일 5분 투자 습관 루프
```

**화면 구성**:
- 예측 질문 3개 (난이도별)
  - 질문 1 (쉬움): "내일 코스피는?" (상승/하락/보합)
  - 질문 2 (중간): "미국 S&P500은?" (상승/하락/보합)
  - 질문 3 (어려움): "외국인 순매수/순매도?" (순매수/순매도)
- 각 질문별 투표 버튼 표시
- 하단: "어제 예측 결과 보기" 버튼

**캡션**:
```
🎮 내일 시장 예측 → 복기 → 기준 형성
매일 5분 투자 습관 루프
```

**촬영 팁**:
- ✅ 질문 3개 모두 보이도록
- ✅ 보상 크레딧 표시 ("적중 시 3C 획득")
- ❌ 이미 투표한 상태 사용 금지 (사용자가 직접 투표하는 느낌 유지)

### 스크린샷 3: AI 진단 & 처방전

**캡처 화면**: `app/(tabs)/diagnosis.tsx` (분석 탭) — AI 진단 결과

**메시지 (상단 오버레이)**:
```
🤖 Gemini AI 기반 건강 점수 분석
A~F 등급 + 리밸런싱 처방전
```

**화면 구성**:
- 건강 점수 게이지 (A 등급, 85점)
- 점수 분석:
  - 자산 배분: "주식 60% + 채권 30% + 현금 10% (양호)"
  - 리스크 수준: "변동성 15% (중간)"
  - 집중도: "삼성전자 30% (높음, 분산 필요)"
- 처방전 버튼: "리밸런싱 처방전 보기"
- 하단: "🤖 Powered by Google Gemini"

**캡션**:
```
🤖 Gemini AI 기반 건강 점수 분석
A~F 등급 + 리밸런싱 처방전
```

**촬영 팁**:
- ✅ A 또는 B 등급 사용 (긍정적 느낌)
- ✅ Gemini 출처 표기 반드시 표시
- ❌ F 등급 사용 금지 (부정적 느낌)

### 스크린샷 4: 건강 점수 추이 (비주얼 피드백)

**캡처 화면**: `app/(tabs)/rebalance.tsx` (처방전 탭) — 건강 점수 히스토리

**메시지 (상단 오버레이)**:
```
📊 내 포트폴리오 건강 점수 추이
90일 히스토리 + 예측 적중 히트맵
```

**화면 구성**:
- 건강 점수 추이 그래프 (90일, C → B → A 상승 추세)
- 예측 적중 히트맵 (GitHub 스타일, 초록색 타일)
- 통계:
  - 평균 건강 점수: B (82점)
  - 예측 적중률: 67% (또래 평균 52% 대비 +15%p)
  - 연속 출석: 37일

**캡션**:
```
📊 내 포트폴리오 건강 점수 추이
90일 히스토리 + 예측 적중 히트맵
```

**촬영 팁**:
- ✅ 상승 추세 그래프 사용 (성장 느낌)
- ✅ 히트맵 밀도 높게 (활발한 사용 느낌)
- ❌ 하락 추세 사용 금지

### 스크린샷 5: 크레딧 경제 & 뱃지

**캡처 화면**: `app/(tabs)/profile.tsx` (전체 탭) — 크레딧 & 뱃지 섹션

**메시지 (상단 오버레이)**:
```
💎 출석 보상 + 예측 적중 크레딧
127일 연속 출석으로 레전드 뱃지 획득
```

**화면 구성**:
- 크레딧 잔액: "52C (₩5,200)" (원화 병기)
- 획득 내역:
  - 출석 보상: +2C (₩200)
  - 예측 적중: +3C (₩300)
  - 공유 보너스: +5C (₩500)
- 뱃지 목록:
  - 🏆 레전드: 365일 연속 출석 (잠김)
  - 💪 철인: 90일 연속 출석 (잠김)
  - 🔥 일주일 전사: 7일 연속 출석 (획득)
  - 📊 분석가: 예측 적중률 60%+ (획득)

**캡션**:
```
💎 출석 보상 + 예측 적중 크레딧
127일 연속 출석으로 레전드 뱃지 획득
```

**촬영 팁**:
- ✅ 뱃지 2~3개 획득 상태 (진행감)
- ✅ 크레딧 원화 병기 반드시 표시 ("52C (₩5,200)")
- ❌ 모든 뱃지 잠금 상태 사용 금지 (동기 부여 X)

## 4. 스크린샷 제작 프로세스

### 4.1 사전 준비 (개발자 작업)

1. **테스트 계정 생성**:
   - 이메일: `test@baln.app`
   - 비밀번호: (내부 공유)
   - 데이터: 실제 데이터로 채워진 계정 (가짜 아님)

2. **화면 데이터 채우기**:
   - 포트폴리오: 삼성전자 30주, 애플 10주, 비트코인 0.05개 등
   - 건강 점수: B 등급 (82점)
   - 예측 기록: 67% 적중률
   - 연속 출석: 37일
   - 크레딧: 52C (₩5,200)
   - 뱃지: 🔥 일주일 전사, 📊 분석가 획득

3. **스크린샷 캡처 도구**:
   - iOS: iPhone 15 Pro Max 실기기 또는 Xcode Simulator
   - Android: Pixel 8 Pro 또는 Android Emulator

---

### 4.2 캡처 방법

**iOS (iPhone 15 Pro Max)**:
```bash
1. 앱 실행 (Expo 개발 빌드)
2. 각 화면으로 이동 (index.tsx → diagnosis.tsx → rebalance.tsx → profile.tsx)
3. 사이드 버튼 + 볼륨 업 버튼 동시 누르기 (스크린샷)
4. 사진 앱에서 확인 → AirDrop으로 Mac 전송
```

**Android (Pixel 8 Pro)**:
```bash
1. 앱 실행 (Expo 개발 빌드)
2. 각 화면으로 이동
3. 전원 버튼 + 볼륨 다운 버튼 동시 누르기 (스크린샷)
4. Google Drive에 업로드
```

3. **iPhone 15 Pro Max** 선택

   - 스크린샷 2: "🎮 내일 시장 예측 → 복기 → 기준 형성"
