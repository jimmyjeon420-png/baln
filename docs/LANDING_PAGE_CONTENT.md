## 3. 솔루션 섹션 (baln의 해결책)

### 3.1 헤드라인
```
✅ baln은 이렇게 돕습니다
```

### 3.2 핵심 기능 4가지

#### 기능 1: 맥락 카드
```
📚 맥락 카드 (Context Card)

오늘 시장이 왜 이렇게 움직였는지
4겹 레이어로 설명합니다:

1. 역사적 맥락: "2008년에도 이런 패턴이 있었고..."
2. 거시경제 체인: "미국 CPI 발표 → 금리 인상 우려 → 기술주 하락"
3. 기관 행동: "외국인 3일 연속 순매도 중 (리밸런싱 시즌)"
4. 내 포트폴리오 영향: "당신의 포트폴리오는 -1.2% 영향"

→ "아, 매크로 때문이구나" (안심)

[스크린샷: 맥락 카드 UI]
```

#### 기능 2: 예측 게임
```
🎮 예측 게임 (Prediction Game)

맥락을 읽은 뒤 내일 시장을 예측:
• 매일 3개 질문 (난이도별)
• 다음날 정답 + 해설 제공
• 적중 시 크레딧 보상 (3C = ₩300)
• 오답도 학습 (복기 노트 작성)

→ 127일 연속 출석 → 패닉셀 확률 80% ↓

[스크린샷: 예측 투표 화면]
```

#### 기능 3: AI 진단
```
🤖 AI 포트폴리오 진단 (Gemini AI)

건강 점수 A~F 등급 분석:
• 자산 배분 밸런스
• 리스크 수준 (변동성, 집중도)
• 리밸런싱 처방전 (언제, 무엇을, 얼마나)

무료: 1회/일 | Premium: 3회/일

[스크린샷: AI 진단 결과]
```

#### 기능 4: 또래 비교
```
📊 또래 비교 (Peer Comparison)

같은 자산 구간 평균과 비교:
"당신은 5,000만원 구간 상위 23%입니다"

⚠️ 주의: 수익률이 아닌 "맥락 이해도"를 비교
(예측 적중률 + 복기 완료율)

MAU 200명 달성 시 자동 오픈 🔒

[스크린샷: 또래 비교 대시보드]
```

## 7. FAQ 섹션

### Q1. baln은 투자 권유 앱인가요?
```
❌ 아니요. baln은 투자 교육 목적 앱입니다.

✅ 제공: 시장 맥락 이해, 투자 지식 학습
❌ 제공 안 함: 종목 추천, 수익 보장, 투자 자문

모든 투자 결정은 본인의 책임입니다.
```

### Q2. 크레딧은 현금으로 환불 가능한가요?
```
❌ 아니요. 크레딧은 앱 내에서만 사용 가능합니다.

1크레딧 = ₩100 상당 (환산 기준)
AI 분석 추가, Premium 체험 등에 사용 가능합니다.
```

### Q3. 무료로 계속 사용할 수 있나요?
```
✅ 네, 맥락 카드 요약, AI 진단 1회/일, 예측 게임은 영구 무료입니다.

Premium은 더 깊은 학습을 원하는 분들을 위한 선택사항입니다.
```

### Q4. 또래 비교는 언제 오픈되나요?
```
MAU 200명 달성 시 자동 오픈됩니다.

현재 진행 상황은 앱에서 실시간으로 확인 가능합니다.
(예: "🔒 200명까지 25% (현재 50명)")
```

### Q5. 계정 삭제는 어떻게 하나요?
```
앱 설정 > 계정 삭제
개인정보는 즉시 삭제됩니다 (30일 이내 완전 삭제).
```

## 12. /security 페이지 (보안 백서)

### 12.1 헤드라인
```
🔐 baln 보안 백서
```

### 12.2 보안 조치
```
1. 암호화
• 비밀번호: bcrypt (Supabase Auth 기본)
• 통신: SSL/TLS (HTTPS)
• 데이터베이스: Supabase 암호화 (AES-256)

2. 접근 제어
• Row-Level Security (RLS): 본인 데이터만 조회 가능
• 최소 권한 원칙: 개발자만 데이터베이스 접근
• 2단계 인증: Supabase Dashboard (OTP)

3. 데이터 보호
• 백업: 일일 자동 백업 (Supabase 기본)
• 재해 복구: AWS 다중 리전 복제
• 모니터링: 실시간 보안 로그 (Sentry)

4. 제3자 서비스
• Supabase: ISO 27001 인증 (AWS 미국 서버)
• Gemini AI: Google LLC (개인 식별 정보 제외)

5. 취약점 대응
• 정기 점검: 분기별 보안 점검
• 버그 바운티: 출시 후 오픈 예정
• 긴급 패치: 발견 즉시 대응
```

### 12.3 인증 계획
```
향후 보안 인증 로드맵:
• ISMS-P 인증: 출시 후 6개월 목표
• ISO 27001 인증: 출시 후 12개월 목표
```
