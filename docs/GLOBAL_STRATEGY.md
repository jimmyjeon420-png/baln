# baln 글로벌 전략: 80억 인구를 위한 로드맵

> 작성: 2026-03-06

## 핵심 통찰: 왜 baln이 글로벌이 될 수 있는가

**보편적 감정을 건드린다.**

- 한국인만 패닉셀 하는 게 아니다
- 미국 Robinhood 유저도, 인도 Zerodha 유저도, 나이지리아 Bamboo 유저도 똑같이 무섭다
- "시장이 -5% 빠졌을 때 열어보는 앱"은 전 세계 어디서나 필요하다

**경쟁자가 없는 포지션:**

| 앱 | 하는 일 | 안 하는 일 |
|---|---|---|
| Robinhood/토스증권 | 매매 | 패닉 방지 |
| Bloomberg/Yahoo Finance | 뉴스 | 맥락 설명 |
| Duolingo | 습관 형성 | 투자 교육 |
| **baln** | 맥락 + 습관 + 안심 | 매매 (안 함) |

baln은 "브로커"가 아니라 **"투자 감정 코치"**다.
이 포지션은 글로벌에서 아무도 안 잡고 있다.

## Phase 0: 리브랜딩 — "투자 앱"에서 "마인드 앱"으로

### 문제
"투자 앱"으로 포지셔닝하면:
- 각국 금융 규제에 걸린다 (FCA, SEC, MAS...)
- 현지 브로커와 경쟁해야 한다
- 시장마다 종목 데이터를 붙여야 한다

### 해법: "Market Mindfulness"
**baln = 투자자를 위한 명상 앱**

Calm이 수면 불안에 했던 것을 baln이 투자 불안에 한다.

- 태그라인: "Don't panic. Understand."
- 카테고리: Health & Fitness 또는 Education (Finance 아님)
- 규제 우회: 매매 기능 없음, 조언 없음, 맥락만 제공

### 이름
"baln"은 글로벌에서도 쓸 수 있다:
- 발음 쉬움 (barn 아님, bahn 느낌)
- 의미 부여 가능: **B**e **A**t **L**east **N**eutral (최소한 중립을 유지하라)
- .app 도메인 이미 보유

## Phase 1: 영어권 진출 (M1~M6)

### 타겟: 미국/영국/호주/캐나다 (4억 명, 투자 인구 ~1.5억)

### 제품 변경 최소화 전략

**바꿔야 하는 것:**
1. 언어 (이미 en.json 90% 완료)
2. 맥락 카드 소스 → S&P 500, NASDAQ, BTC 중심 (이미 지원)
3. 예측 게임 → 영어 질문 (이미 fallback 있음)
4. 통화 표시 → USD

**안 바꿔도 되는 것:**
- 습관 루프 구조 (보편적)
- 구루 캐릭터 (버핏/달리오는 미국인이 더 잘 안다)
- 건강 점수 시스템 (숫자는 만국 공통)
- 크레딧/보상 시스템

### 핵심 성장 엔진: "Panic Moment Marketing"

**시장이 급락하는 순간이 baln의 마케팅 기회다.**

전략:
1. VIX가 25 이상 → 자동 트리거
2. X(Twitter)에 자동 게시: "S&P dropped 3%. Here's what's actually happening." + 맥락 카드 이미지
3. 맥락 카드를 무료로 공개 (바이럴)
4. "Want this every day? Download baln." CTA

이건 Duolingo의 "Miss a day?" 전략과 같다.
차이점: Duolingo는 죄책감을 쓰고, baln은 **공포의 순간**을 쓴다.

### 성장 채널 우선순위

| 채널 | 비용 | 기대 효과 | 우선순위 |
|------|------|----------|---------|
| X/Twitter 패닉 카드 | $0 | 높음 (바이럴) | P0 |
| Reddit r/investing, r/stocks | $0 | 높음 (커뮤니티) | P0 |
| TikTok "market crash explained in 60s" | $0 | 매우 높음 | P0 |
| Product Hunt 런칭 | $0 | 중간 (초기 트래픽) | P1 |
| 유튜브 쇼츠 | $0 | 높음 | P1 |
| Apple Search Ads | $50/일 | 중간 | P2 |

### M6 목표
- MAU 10,000 (미국 중심)
- 유료 전환 3% → 300명 x $4.99 = $1,497/월
- App Store 평점 4.5+

## Phase 3: 플랫폼화 (M12~M24)

### "Market Mindfulness" 플랫폼

baln이 직접 콘텐츠를 만드는 대신, **구조를 제공**한다.

#### 3-1. 현지 구루 시스템
- 각국의 투자 인플루언서가 "구루"로 입점
- 인도: Nithin Kamath (Zerodha 창업자) 스타일 AI 구루
- 일본: 탈려맨 (유명 투자 블로거) 스타일 AI 구루
- 브라질: Thiago Nigro (O Primo Rico) 스타일
- 수익 분배: 구루 30% / baln 70%

#### 3-2. 브로커 제휴 (수수료 모델)
- baln에서 "이 종목 사고 싶다" → 현지 브로커 앱으로 딥링크
- 제휴 수수료: 거래당 $0.10~$1.00
- 미국: Robinhood/Webull, 인도: Zerodha/Groww, 한국: 토스증권/키움
- baln은 매매 기능을 절대 직접 만들지 않는다 (규제 회피)

#### 3-3. B2B: 브로커에게 "안심 엔진" 판매
- 토스증권/Robinhood 같은 브로커가 자기 앱에 baln의 맥락 카드를 임베드
- API 라이선스: $0.01/사용자/일
- 100만 사용자 브로커 = $10,000/일 = $300,000/월

## 경쟁 해자 (Moat): 왜 baln을 카피할 수 없나

### 1. 데이터 플라이휠
```
사용자 예측 데이터 축적
→ "집단 심리 지표" 생성 (baln만의 독점 데이터)
→ "지금 baln 사용자 78%가 하락을 예측합니다" = 역지표
→ 이 데이터가 맥락 카드를 더 정확하게 만듦
→ 더 많은 사용자 유입
```

### 2. 습관 잠금
- 200일 연속 출석 사용자는 절대 이탈 안 함
- Duolingo와 같은 원리: 스트릭이 감옥이 된다

### 3. 구루 생태계
- 10명 → 100명 → 1,000명 AI 구루
- 각국 현지 투자 인플루언서 = 네트워크 효과
- 경쟁자가 구루 1명을 만들 동안 baln은 100명

### 4. "패닉 순간" 브랜드 인지
- 시장이 급락할 때마다 baln 로고가 SNS에 뜬다
- "시장 떨어졌을 때 baln 봤어?" = 입소문 엔진
- 이건 시간이 쌓여야 만들어지는 브랜드 자산

## 부록: 80억 인구 도달 경로

```
한국 (5,000만) ← 지금 여기
  + 미국/영국/호주 (4억)
  + 인도 (14억)
  + 동남아 (7억)
  + 일본 (1.3억)
  + 유럽 (4.5억)
  + 중남미 (6.5억)
  + 아프리카 (14억)
  + 중동 (4억)
  = ~52억 (스마트폰 보유 인구)

이 중 투자/저축 활동 인구: ~15억
이 중 "시장 불안을 느끼는" 인구: ~5억
이 중 앱을 설치할 인구: ~5,000만 (TAM)
이 중 유료 전환: ~150만 (SAM)

150만 x $3.50/월 x 12개월 = $63M/년 (약 840억원)
```

이게 baln의 천장이다. 한국만으로는 절대 도달할 수 없는 숫자.
영어 하나 추가하는 순간, 시장이 100배가 된다.
