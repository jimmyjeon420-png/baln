## 1. 법적 배경

### 1.1 자본시장법 제335조 (무등록 영업행위의 금지)

```
제335조(무등록 영업행위의 금지)
① 금융위원회의 인가를 받지 아니한 자는 제6조 제1항 각 호의 금융투자업을 영위하여서는 아니 된다.

제6조 제5호 — 투자자문업: 금융투자상품의 가치 또는 금융투자상품에 대한 투자판단(종목, 종류, 취득·처분, 취득·처분의 방법·수량·가격 및 시기 등에 대한 판단을 말한다)에 관한 자문에 응하는 것을 영업으로 하는 것

벌칙: 5년 이하의 징역 또는 2억원 이하의 벌금 (제445조)
```

**핵심**: 특정 종목을 추천하거나, 매수/매도 타이밍을 조언하는 행위는 **무등록 투자자문업**에 해당.

## 2. baln 앱의 법적 포지셔닝

### 2.1 baln은 "투자 교육 앱"이다

| 항목 | 투자자문업 (금지) | baln 앱 (합법) |
|------|------------------|----------------|
| **주체** | 투자자문업 인가 필요 | 교육 서비스 제공 (인가 불요) |
| **활동** | 특정 종목 매수/매도 권유 | 시장 맥락 이해 교육 |
| **대가** | 자문료 수취 | 교육 콘텐츠 구독료 (Premium) |
| **책임** | 투자 손실 책임 일부 부담 가능 | 투자 손실 책임 없음 (면책 조항) |

### 2.2 baln이 제공하는 것 (합법)

✅ **교육 콘텐츠**:
- 시장 맥락 카드 ("오늘 미국 CPI가 발표되었습니다")
- 거시경제 체인 설명 ("금리 인상 → 기술주 하락 → 삼성전자 연동")
- 역사적 맥락 ("2008년에도 이런 패턴이 있었습니다")

✅ **일반적인 가이드**:
- 자산 배분 원칙 ("주식 60% + 채권 40% 분산")
- 리밸런싱 시기 ("분기 1회 리밸런싱 권장")
- 리스크 관리 ("변동성이 높으면 채권 비중 늘리기")

✅ **AI 분석 (참고용)**:
- 포트폴리오 건강 점수 (A~F 등급)
- 자산 배분 밸런스 분석
- 일반적인 리밸런싱 처방전 ("주식 비중이 70%로 높습니다. 60%로 조정 고려")

### 2.3 baln이 제공하지 않는 것 (금지)

❌ **종목 추천**:
- "삼성전자를 지금 매수하세요" (X)
- "카카오를 내일 매도하세요" (X)
- "테슬라가 오를 것 같습니다" (X)

❌ **매매 타이밍 조언**:
- "지금이 매수 타이밍입니다" (X)
- "내일 오전에 매도하세요" (X)
- "이번 주 안에 청산하세요" (X)

❌ **수익 보장**:
- "이 전략으로 연 10% 수익 보장" (X)
- "원금 보장 포트폴리오" (X)
- "확정 수익률 제공" (X)

## 3. 금지 표현 체크리스트

### 3.1 코드 검색 결과 (2026-02-11)

**검색어**: "수익 보장", "종목 추천", "매수하세요", "매도하세요", "오를 것", "떨어질 것"

**결과**: ✅ **모두 깨끗함** (금지 표현 0건)
- "수익 보장" 출현: 2건 → 모두 "금지" 맥락 (이용약관, 면책 조항)
- 기타 금지 표현: 0건

**검색 실행 로그**:
```bash
# 2026-02-11 실행
grep -r "수익 보장" app/ src/ docs/
# → 2건 발견 (모두 "수익 보장 안 함" 맥락)

grep -r "종목 추천" app/ src/ docs/
# → 0건

grep -r "매수하세요\|매도하세요" app/ src/ docs/
# → 0건
```

### 3.2 허용 표현 vs 금지 표현 가이드

| 허용 (교육) | 금지 (자문) |
|------------|------------|
| "일반적으로 주식 60% 분산 권장" | "삼성전자 60% 보유하세요" |
| "변동성이 높으면 채권 비중 고려" | "국채를 지금 매수하세요" |
| "2008년 금융위기 당시 패턴" | "지금이 2008년과 같으니 매도하세요" |
| "외국인 순매도 중입니다" | "외국인이 팔고 있으니 따라 파세요" |
| "건강 점수 C등급 (리스크 높음)" | "C등급이니 지금 매도하세요" |

## 4. 커뮤니티 금지 행위 (사용자 간)

### 4.1 커뮤니티 규칙 (자본시장법 준수)

baln 커뮤니티(VIP 라운지)는 **투자 경험 공유 공간**이며, 다음 행위는 금지됩니다:

❌ **종목 추천** (자본시장법 제335조 위반):
- "삼성전자 지금 사세요"
- "카카오가 오를 것 같아요. 매수 추천"
- "○○주식 매수 타이밍입니다"

❌ **수익 보장 표현** (자본시장법 제50조 위반):
- "이 전략으로 확정 수익 10%"
- "원금 보장 포트폴리오"
- "100% 수익 가능"

❌ **외부 유인** (무등록 투자자문업 우려):
- "텔레그램으로 오세요. 종목 알려드려요"
- "카톡방에서 무료 자문해드립니다"
- "유료 방 가입하면 종목 추천"

✅ **허용 표현** (경험 공유):
- "저는 삼성전자를 20% 보유하고 있어요" (본인 경험)
- "어제 애플 실적 발표가 있었네요" (사실 공유)
- "저는 이런 기준으로 투자해요" (철학 공유)

### 4.2 위반 시 조치

| 위반 횟수 | 조치 |
|----------|------|
| 1회 | 경고 메시지 + 게시글 삭제 |
| 2회 | 7일 커뮤니티 정지 |
| 3회 | 영구 계정 정지 + 수사기관 통보 (자본시장법 위반 우려) |

## 7. Gemini AI 분석 면책

### 7.1 AI 분석 결과 안내

baln 앱은 Google Gemini AI를 사용하여 포트폴리오 건강 점수를 분석하지만, 이는 **참고용 정보**입니다.

**AI 분석 화면 하단 표시**:
```
━━━━━━━━━━━━━━━━━━━━━━━━━━
ℹ️ AI 진단 결과 안내
━━━━━━━━━━━━━━━━━━━━━━━━━━

이 분석 결과는 Gemini AI 기반으로 생성된 참고용 정보입니다.

✅ 분석 범위:
- 포트폴리오 구성 밸런스 (자산 배분)
- 리스크 수준 (변동성, 집중도)
- 일반적인 리밸런싱 가이드

❌ 분석 제외:
- 개별 종목의 미래 가격 예측
- 매수/매도 타이밍 추천
- 수익률 보장

⚠️ 주의사항:
AI는 과거 데이터를 기반으로 분석하며, 미래를 예측할 수 없습니다.
실제 투자 결정은 본인의 판단과 책임 하에 이루어져야 합니다.

전문가 상담이 필요한 경우 금융투자협회(1332) 또는
한국투자자보호재단(1644-8485)에 문의하세요.
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 8. 인가 불요 확인 (교육 서비스)

### 8.1 baln이 인가를 받지 않아도 되는 이유

**금융위원회 유권해석 (2018-금융투자-001)**:
```
「자본시장과 금융투자업에 관한 법률」 제6조 제5호의 투자자문업은
"금융투자상품의 가치 또는 금융투자상품에 대한 투자판단에 관한 자문"을
영업으로 하는 경우에 한정됩니다.

단순히 일반적인 투자 교육 콘텐츠를 제공하거나,
투자 관련 정보를 제공하는 행위는 투자자문업에 해당하지 않습니다.
```

**baln 앱의 포지셔닝**:
- ✅ 일반적인 투자 교육 콘텐츠 제공 (맥락 카드, 거시경제 설명)
- ✅ 투자 지식 학습 도구 (예측 게임, 복기)
- ✅ 참고용 AI 분석 (건강 점수, 리밸런싱 가이드)
- ❌ 특정 종목에 대한 매수/매도 권유 (X)
- ❌ 개별 맞춤 투자 자문 (X)

**결론**: baln은 투자자문업 인가가 필요하지 않습니다.
