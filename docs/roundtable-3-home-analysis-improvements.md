## 📋 사용자 피드백 요약

### 1. 홈 화면 (index.tsx) 문제
- ❌ "오늘의 시장 분석" → "분석 중"만 표시, 결과 없음
- ❌ 퀴즈 카드 → 단순 베팅 느낌, 오를/내릴 **근거를 뉴스로** 알려줘야 함

### 2. 분석 화면 (rebalance.tsx) 문제
- ❌ 패닉 실드 72점 → **왜 이런 점수인지 전혀 알 수 없음**
- ❌ 건강 점수 구성 요소 → 클릭해도 설명이 이해 안 됨
  - 왜 중요한지?
  - 사례는 무엇인지?
  - 배분 이탈도를 왜 하는지, 하면 뭐가 좋은지?
- ❌ AI 심화 분석 → 4개 섹션 모두 동작 안 함

### 3. 투자 감정 기능 (새 제안)
- ✨ 그날그날 감정 기록 + 히스토리 보기

- ✨ 다른 사람들의 감정/투자일기 보기
- ✨ 등급별 감정 차이 보기

## 💬 라운드테이블 토론

### 이승건 (토스 대표) — "설명 없는 점수는 무용지물"

#### 핵심 의견
> "패닉 실드 72점이라고 떡하니 써놨는데, 왜 72점인지 모르면 유저는 그냥 지나간다. **점수 옆에 한 줄 이유**를 꼭 붙여라."

#### 구체적 제안

**1. 점수 → 이유 패턴**
```
❌ Before: "패닉 실드 72점"
✅ After: "패닉 실드 72점 — 현금 비중 20%로 안정적"
```

**2. 건강 점수 구성 요소 → 스토리텔링**
```
❌ Before: "배분 이탈도: 12% (나쁨)"
✅ After: "삼성전자가 목표보다 5%p 많아요. 3주 내 조정 권장"
```

**3. 퀴즈 → 교육 콘텐츠로 업그레이드**
```
❌ Before: "금리가 오를까요?"
✅ After:
"금리가 오를까요?"
📰 오를 근거: CPI 3개월 연속 상승 (블룸버그)
📰 내릴 근거: 실업률 4.2% → 경기 둔화 우려 (WSJ)
```

#### 우선순위
- **P0 (즉시)**: 점수 옆에 이유 한 줄 추가
- **P0 (즉시)**: 퀴즈에 뉴스 근거 2개 추가
- **P1 (2주)**: 건강 점수 구성 요소 스토리텔링
- **P2 (출시 후)**: 투자 감정 일기 + 커뮤니티

---

### 레이 달리오 — "맥락 없는 데이터는 공포를 만든다"

#### 핵심 의견
> "사용자가 '패닉 실드 72점'을 보고 불안해한다면, 맥락이 부족한 거다. **72점의 역사적 의미**를 알려줘야 한다."

#### 구체적 제안

**1. 역사적 맥락 추가**
```
패닉 실드 72점
→ "2008년 금융위기 때는 30점이었어요. 지금은 안전한 구간입니다."
```

**2. 또래 비교 (Peer Context)**
```
배분 이탈도 12%
→ "같은 자산 구간 평균은 18%예요. 당신은 평균보다 잘하고 있어요."
```

**3. AI 심화 분석 → 시나리오 플래너**
```
현재 문제: 4개 섹션이 동작 안 함
달리오 제안: "What-If 시나리오"를 보여줘라
- "만약 금리가 0.5%p 오르면? → 당신 포트폴리오 -2.3% 예상"
- "만약 삼성전자를 10% 팔면? → 건강 점수 83점 → 95점"
```

#### 우선순위
- **P0 (즉시)**: 점수에 역사적 맥락 추가 (한 줄)
- **P1 (2주)**: 또래 비교 추가
- **P1 (2주)**: What-If 시나리오 1개 작동 (금리 변동)
- **P2 (출시 후)**: 나머지 3개 시나리오

---

### 워렌 버핏 — "복잡한 건 안 본다. 심플하게."

#### 핵심 의견
> "AI 심화 분석 4개가 다 안 된다면, **차라리 1개만 완벽하게** 만들어라. 나머지는 '준비 중'이라고 솔직히 말해라."

#### 구체적 제안

**1. 기능 축소 → 완성도 높이기**
```
현재: AI 심화 분석 4개 (모두 동작 안 함)
버핏 제안: 1개만 먼저 완성
- "배분 최적화" → 완벽하게 작동
- 나머지 3개 → "준비 중 (2월 말 공개)"
```

**2. 투자 감정 일기 → "장기 투자" 습관 만들기**
```
버핏: "감정 일기를 쓰면, 공포 때 판 걸 나중에 후회하게 된다. 좋은 교육이다."

기능:
- 매일 감정 + 메모 (30자)
- 30일 후 리마인더: "한 달 전 당신은 '공포'였는데, 지금 자산은 +8%예요"
```

**3. 퀴즈 → "장기 사고" 유도**
```
❌ Before: "내일 코스피가 오를까요?"
✅ After: "3개월 후 금리가 내릴까요?" (단기 투기 방지)
```

#### 우선순위
- **P0 (즉시)**: AI 심화 분석 → 1개만 작동, 나머지 "준비 중" 표시
- **P1 (2주)**: 투자 감정 일기 MVP (감정 + 메모 30자)
- **P2 (출시 후)**: 감정 일기 커뮤니티 공유
