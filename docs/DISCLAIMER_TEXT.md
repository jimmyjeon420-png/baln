# 면책 조항 텍스트 (Disclaimer)

**목적**: 앱 내 온보딩 및 AI 진단 화면에 표시할 면책 조항
**사용 위치**:
- 온보딩 마지막 화면
- AI 진단 결과 화면
- 맥락 카드 하단
- 이용약관 제10조

## 1. 온보딩 화면 면책 조항

### 1.1 짧은 버전 (1줄, 온보딩 최하단)
```
이 앱은 투자 교육 목적이며, 투자 권유가 아닙니다. 모든 투자 결정은 본인의 책임입니다.
```

**표시 위치**: `app/onboarding/` 마지막 화면 (회원가입 완료 후)
**스타일**: 작은 회색 텍스트 (10pt, 회색 #666)

---

### 1.2 긴 버전 (온보딩 전용 화면, 선택사항)

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
⚠️ 중요한 안내사항
━━━━━━━━━━━━━━━━━━━━━━━━━━

baln 앱은 투자 교육 목적 앱입니다.

✅ 제공하는 것:
- 시장 맥락 이해 (맥락 카드)
- 포트폴리오 건강 점수 (AI 진단)
- 투자 지식 학습 (예측 게임)

❌ 제공하지 않는 것:
- 종목 추천 (특정 주식 매수/매도 권유)
- 투자 자문 (개별 상담)
- 수익 보장 (원금 보장, 확정 수익률)

📌 모든 투자 결정은 본인의 책임입니다.
투자 전 반드시 전문가와 상담하시기 바랍니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**표시 위치**: 온보딩 3단계 중 마지막 (선택사항)
**UI**: 모달 팝업 또는 전체 화면, "이해했습니다" 버튼

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
ℹ️ AI 진단 결과 안내
━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 분석 범위:
- 포트폴리오 구성 밸런스 (자산 배분)

AI는 과거 데이터를 기반으로 분석하며, 미래를 예측할 수 없습니다.
실제 투자 결정은 본인의 판단과 책임 하에 이루어져야 합니다.

전문가 상담이 필요한 경우 금융투자협회(1332) 또는
한국투자자보호재단(1644-8485)에 문의하세요.

━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**표시 위치**: AI 진단 상세 화면 또는 "분석 결과 정보" 버튼 클릭 시
**UI**: 모달 팝업

## 4. 커뮤니티 게시판 안내문

### 4.1 게시판 상단 고정 공지
```
━━━━━━━━━━━━━━━━━━━━━━━━━━
⚠️ 커뮤니티 이용 규칙
━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 허용:
- 본인의 투자 경험 공유 ("저는 삼성전자를 매수했습니다")
- 시장 뉴스 공유 ("애플 실적 발표가 곧 있어요")
- 투자 철학 공유 ("저는 이런 기준으로 투자합니다")

❌ 금지 (계정 정지 대상):
- 종목 추천 ("삼성전자 지금 사세요")
- 수익 보장 ("원금 보장", "확정 수익률 10%")
- 외부 유인 ("텔레그램으로 오세요", "카톡방 초대")
- 대가 요구 ("종목 알려드릴게요 (유료)")

⚖️ 법적 근거:
자본시장법 제335조 - 무등록 투자자문업 시 5년 이하 징역 또는 2억원 이하 벌금

위반 시 즉시 계정 정지 및 수사기관 통보됩니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**표시 위치**: `app/(tabs)/lounge.tsx` 상단 고정
**UI**: 접을 수 있는 배너 (처음 1회만 전체 표시, 이후 축소)

## 5. 이용약관 제10조 (법적 문서용)

```
제10조 (책임 제한)

1. 투자 손실 책임 면제

baln 앱은 투자 교육 목적 앱이며, 투자 결과에 대해 어떠한 책임도 지지 않습니다.

- 맥락 카드: 시장 맥락 이해를 돕는 교육 자료일 뿐, 투자 권유가 아닙니다.
- AI 진단: 참고용 분석이며, 실제 투자 결정은 사용자 본인의 책임입니다.
- 예측 게임: 학습 도구일 뿐, 예측 결과가 실제 시장 움직임을 보장하지 않습니다.
- 또래 비교: 익명 통계 자료일 뿐, 투자 성과를 보장하지 않습니다.

투자 결정 전 반드시 전문가와 상담하시기 바랍니다.
```

**표시 위치**: `docs/TERMS_OF_SERVICE.md` 제10조
